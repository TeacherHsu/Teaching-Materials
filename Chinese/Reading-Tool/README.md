國小課文點讀小幫手

一個免安裝、免登入、可直接使用的國小課文點讀工具，
協助學生進行「看字、聽音、跟讀」練習。

(建議：您之後可以截一張軟體執行畫面替換上面的圖片)

✨ 功能特色

🔤 自動注音：輸入中文字即自動標示注音，並支援手動校正（破音字）。

🔊 點字即朗讀：支援瀏覽器原生 TTS 語音，無需下載額外檔案。

🧩 三種閱讀模式：提供「讀字」、「讀詞」、「讀句」三種層次練習。

📝 教師備課：老師可自行編輯課文、校正發音，並一鍵產生分享連結。

📱 跨平台支援：手機、平板、電腦皆可使用（RWD 響應式設計）。

🔐 安全隱私：純前端運作，無帳號、無後端資料庫，不蒐集任何學生個資。

🧑‍🏫 適用對象

國小教師（特教班、資源班、普通班）

學習障礙、閱讀困難的學生

需要在家進行跟讀練習的學生與家長

🚀 使用方式

開啟網頁：直接點擊 GitHub Pages 網址（或打開 HTML 檔案）。

貼上課文：將要練習的課文內容貼入編輯框。

校正注音：

點擊「校正」按鈕。

點擊需要修改的字，輸入正確注音（如破音字）。

若發音不準，可設定「發音借代字」（例如輸入「義」來讀「ㄧˋ」）。

產生連結：點擊「產生連結」，將網址複製分享給學生或家長。

🧩 詞塊標記說明

在「讀詞」模式下，系統需要知道詞彙的邊界。

手動標記（推薦）：
使用 | 符號將詞彙包起來。

範例：我們|一起|去|上學|。
(系統會自動隱藏 | 符號，並以標記範圍進行朗讀)

自動標記：
若未手動標記，系統會自動以標點符號與基本規則進行粗略斷詞。

⚠️ 注意事項

隱私提醒：分享連結中包含了完整的課文內容編碼，請勿在課文中放入學生個資或機密資訊。

發音來源：本工具使用使用者裝置（手機/電腦）內建的語音合成引擎，不同裝置聽起來可能會略有差異（建議優先使用內建台灣繁體中文語音的裝置）。

注音準確度：注音為系統自動運算產生，雖已最佳化，但在多音字情境下仍可能誤判，建議教師務必依教學需求進行校正。
